# Link

```js
import Link from '@symphony/design-system/components/Link';
```

Link is an `anchor` tag, wrapping a `Text` component.

props affecting anchor:

`inNewTab`: `true`/ `false`

`href`

`className`

props affecting Text:

`variant` supports: `h1`, `h2`, `h3`, `h4`, `h5`, `h6`, `subtitle1`, `subtitle2`, `subtitle3`, `body1`, `body2`, `caption`, `overline`.

`color` supports: `light`, `regular`, `primary`, `error`, `gray`, `warning`,

`hoverColor` supports: `light`, `regular`, `primary`, `error`, `gray`, `warning`,

`textClassName`
